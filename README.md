# JDM Rush Imports - Import Cost Calculator

A web-based calculator that helps customers estimate the total cost of importing JDM vehicles to Canada.

## Features

- Live exchange rates from Bank of Canada API
- Calculates all fees: shipping, duties, GST, customs, transport
- Supports multiple destination cities across Canada
- Shows PST separately (paid at registration)
- Mobile responsive design

## Deployment

This project is deployed on Netlify and embedded on the JDM Rush Imports website.
